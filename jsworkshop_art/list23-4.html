Scrolling Window Example

This example shows a scrolling window created in DHTML. The window is actually a layer that shows a portion of a larger layer.

This is the first paragraph of the scrolling message. The message is created with ordinary HTML.

Entries within the scrolling area can use any HTML tags. They can contain Links.

There's no limit on the number of paragraphs that you can include here. They don't even need to be formatted as paragraphs.

  • For example, you could format items using a bulleted list.

The scrolling ends when the last part of the scrolling text is on the screen. You've reached the end.
